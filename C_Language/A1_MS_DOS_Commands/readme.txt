To enter into the specific drive
drive_name:
example: E: or e:

1. copy con - it is used to create a File
syntax: copy con <file_name>
to save and quit - f6 and ctrl + z.

Note: If the file is existing then window will prompt for overwrite

2. To see the list of files command - dir
syntax: dir <directory_name>

3. If you want to see / open any file
syntax: type <file_name>

4. Delete a file
syntax: del <file_name>
If you want to delete a file with permissions command
syntax: del <file_name>/p (small p)

5. Copy a file
If you want to duplicate a file
syntax: copy <source_file_name> <destination_file_name>

6. Rename a file or directory
syntax: rename/ren <source_file_name> <destination_file_name>

7. date/time command
To display current date and time. we can modify also.

8. To check the version - ver

9. If you want to help of any command info
syntax: help <command_name> / command_name/?

10. To exit terminal
syntax: exit

-------------------- MS-DOS Directory Command ---------------------------------------

1. cls -clear the screen

2. To create a directory - md / mkdir

3. To change directory - cd / chdir <directory_name>

4. cd .. - one step back

5. cd / - root

6. check the structure the command - tree
it will show directories only

7. tree/f - it will show all directories and files

8. rd - remove only empty directories
rd/s remove directories along with sub-folders

---------------- Wild Characters ----------------
These are used to filter the files and directories

---> * and ? are wild characters
* - represents group of characters
? - represents single characters
